# QtStuSys - 学生管理系统

QtStuSys 是一个基于 Qt 框架开发的学生管理系统，提供学生信息、课程、成绩的综合管理功能，界面简洁直观，操作便捷。

## 功能模块

系统包含以下核心功能模块：

- **学生管理**
  - 学生信息列表展示
  - 新增学生信息
  - 学生信息批量导入/导出

- **课程管理**
  - 课程信息列表展示
  - 新增课程信息
  - 选课管理（基于中间表操作）

- **成绩管理**
  - 成绩录入（支持按课程/班级录入）
  - 成绩查询与修改
  - 成绩批量导入/导出
  - 成绩作废与恢复

- **系统管理**
  - 操作日志查看
  - 角色权限设置（可选功能）
  - 系统配置

## 技术栈

- **框架**：Qt 6.10.0
- **编译器**：MinGW 64-bit
- **数据库**：通过 DatabaseManager 实现数据库交互（具体数据库类型可配置）
- **UI 样式**：支持 QSS 样式表自定义界面风格

## 编译与运行

1. 环境要求：
   - Qt 6.10.0 及以上版本
   - MinGW 64-bit 编译器
   - 支持 C++11 及以上标准

2. 编译步骤：
   - 使用 Qt Creator 打开项目文件
   - 选择合适的构建套件（Desktop Qt 6.10.0 MinGW 64-bit）
   - 构建项目（Ctrl+B）

3. 运行：
   - 直接在 Qt Creator 中运行（Ctrl+R）
   - 或在 build 目录下找到可执行文件运行

## 核心功能实现说明

### 学生管理模块

- 采用面向对象设计，`Student` 类封装学生信息及数据库操作
- 支持学生信息的增删改查（CRUD）操作
- 实现分页查询和条件查询（按班级/姓名模糊查询）
- 采用逻辑删除（`is_deleted` 标记）机制，保留数据历史

```cpp
// 学生信息查询示例
QList<Student> students = Student::queryAll(1, 20);  // 分页查询
Student student = Student::queryByStudentId("2023001");  // 按学号查询
QList<Student> filtered = Student::queryByCondition("计算机一班", "张");  // 条件查询
```

### 界面设计

- 采用 QWidget 和 QLayout 进行界面布局
- 学生列表使用 QTableWidget 展示
- 支持搜索、刷新、导入导出等常用操作
